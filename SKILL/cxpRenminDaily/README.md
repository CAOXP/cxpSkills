# cxpRenminDaily 🗞️

> **一台通往历史与今天的“时光读报机”。**
>
> 这个 Skill 能帮你快速访问人民网图文数据库（`data.people.com.cn`），不仅能为你提炼今天的人民日报要闻，更能带你穿越回“历史上的今天”，看看当年的头版头条都在关心什么。

---

## ✨ 这个 Skill 能为你做什么？

1. **📰 今日要闻速递**
   没时间看报纸？一句话触发，自动抓取当天《人民日报》第1版内容，并为你提炼出结构化、易阅读的头条摘要卡片。

2. **🕰️ 历史上的今天（时光机）**
   好奇十年前的今天发生了什么大事？它能带你翻开历史的旧报纸，看看 2008 年奥运开幕当天、或是 2001 年申奥成功次日的头条。

3. **📅 特定日期查询**
   你可以精确指定某年某月某日，只要数据库里有，它就能把那一天的头版送到你面前。

---

## 🚀 极速上手

在支持 Skill 的 AI IDE 中，直接输入以下指令触发：

### 场景一：看今天的报纸
```text
看看今天的人民日报头条。
```

### 场景二：穿越历史上的今天
```text
历史上的今天，人民日报头条是什么？
```
*(AI 会自动提取当前日期的月和日，并带你回顾过去重要年份的今天)*

### 场景三：精确日期查询
```text
帮我查一下 2008年8月8日 的人民日报第1版。
```

## ⚙️ 它是怎么工作的？

本技能强制绑定了权威的**人民网图文数据库**进行检索。

**核心 URL 结构：**
`https://data.people.com.cn/rmrb/{YYYYMMDD}/1`

AI 会自动执行以下流程：
1. 识别并解析你的日期意图。
2. 拼接出对应的数据库 URL（默认访问第 1 版：要闻）。
3. 抓取该版面的所有新闻标题。
4. 过滤掉无意义的版面声明，提取出核心事件或政策导向。
5. 输出一张排版精美的 Markdown “读报卡片”。

## 📝 输出示例

```markdown
📰 **人民日报要闻摘要**
📅 **日期**：2026年4月3日 | 📍 **版面**：第1版 (要闻)

---

### 🌟 头条导读
- **[新疆将资源优势转化为高质量发展胜势（“十五五”开好局起好步）]**
  - *摘要：报道新疆在“十五五”开局阶段如何利用自身资源优势推动高质量发展的重要举措与成果。*
- **[在雄安，有群塞罕坝的造林人]**
  - *摘要：讲述雄安新区建设中，传承和发扬塞罕坝精神的造林人的奋斗故事。*

### 📌 其它重要简讯
- 读懂通道宽畅背后的暖意（今日谈）
- 导弹为“刃”，日本将自己推上危险的刀锋（钟声）
- 形成合力，提升养老服务能力（专家点评）
- 守本分，怀“痴情”（人民论坛）
- “一步一步”里的中国力量（评论员观察）

---
🔗 [查看原报纸版面](https://data.people.com.cn/rmrb/20260403/1)
```

---

## ⚠️ 注意事项与核心原则

- **绝对忠于原文**：本技能绝不编造新闻，不篡改标题，所有信息均来自网页抓取。
- **遇到查无此日**：如果某些早期节假日无报纸，或者当天报纸尚未更新，AI 会如实告知“该日期暂无数据”。
- 本技能主要关注**第 1 版（要闻）**，以确保提取到的都是最具分量的新闻。
