## 推荐用法

```text
我先按 PATENT_TEMPLATE.md 填了一版。
请你以此为基础：
1. 建立问题 → 方法 → 技术扩展 → 系统整合 → 效果主线
2. 按方法/技术/系统三层提炼保护点
3. 补出递进实施例
4. 对缺失处统一标“待补充：”并提出精准问题
```
